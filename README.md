# Superstore Sales Analytics Dashboard

This project demonstrates an end-to-end data analysis workflow using Python, SQL, and Power BI.

## Tools Used
- Python (Pandas)
- PostgreSQL (SQL)
- Power BI

## Key Insights
- Total Sales: 2.26M
- Total Orders: 9800
- Total Customers: 793

## Features
- Sales Trend Analysis
- Region Performance
- Product Analysis
- Category Distribution
